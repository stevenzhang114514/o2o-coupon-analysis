## 二、核销率分析

### 2.1 按优惠券类型

| 券类型 | 发放量 | 核销量 | 核销率 |
|--------|--------|--------|--------|
| 满减券 | 1,760,821 | 156,442 | **8.88%** |
| 折扣券 | 33,272 | 3,853 | **11.58%** |
| 无券消费 | 164,818 | 135,399 | 82.15% |

**发现**：折扣券核销率略高于满减券（+2.7pp），但满减券的绝对发放量是折扣券的 53 倍，是平台主流券种。

### 2.2 按满减门槛

| 门槛 | 发放量 | 核销率 |
|------|--------|--------|
| ≤20元 | 338,687 | **14.42%** |
| 21-50元 | 611,907 | 9.47% |
| 51-100元 | 338,109 | 5.54% |
| 101-200元 | 315,336 | 6.03% |
| 201-500元 | 133,162 | 7.21% |
| >500元 | 23,620 | 9.88% |

**发现**：门槛越低核销率越高，≤20元满减券核销率最高（14.42%）。商户推券应尽量放低门槛。

### 2.3 按距离

| 距离 | 核销率 |
|------|--------|
| 0m（同位置） | **13.46%** |
| 1m | 5.98% |
| 2-3m | 3.92% |
| 4-5m | 2.96% |
| 6-10m | 1.72% |
| 10m+ | 未统计 |

**发现**：距离是核销率的第一影响因素。0m 核销率是 10m+ 的约 **7.8 倍**。推送优惠券时应优先覆盖附近用户。

## 三、时间维度

### 3.1 月度核销趋势

| 月份 | 核销量 | 活跃用户 | 人均核销 |
|------|--------|----------|----------|
| 2016-01 | 1,940,071 | 371,150 | 5.23 |
| 2016-02 | 1,116,562 | 274,165 | 4.07 |
| 2016-03 | 2,093,507 | 409,701 | 5.11 |
| 2016-04 | 1,952,447 | 396,041 | 4.93 |
| 2016-05 | 2,131,073 | 429,124 | 4.97 |
| 2016-06 | 2,317,252 | 420,606 | 5.51 |

**发现**：
- 二月因**春节**导致核销量骤降（仅 112 万，比 1 月少 42%）
- 6 月是核销高峰（232 万），1→6 月整体增长 **+19.6%**
- 用户人均月核销 4~5.5 次，粘性较好

### 3.2 工作日 vs 周末

| 日期类型 | 核销量 | 日均核销 |
|----------|--------|----------|
| 工作日 | 8,683,323 | **66,795** |
| 周末 | 2,867,589 | 55,146 |

**发现**：工作日是核销主力，日均比周末高 21%。优惠券推送可侧重工作日。

### 3.3 领券-核销时差

| 时差 | 占比 |
|------|------|
| **当天核销** | **10.32%** |
| 次日核销 | 0.87% |
| 2-3天 | 0.96% |
| 4-7天 | 1.24% |
| 8-15天 | 1.01% |
| 16-30天 | 0.62% |
| >30天 | 0.13% |
| 未核销 | **84.84%** |

**发现**：
- **84.84% 的券领后从未使用**，核销转化空间巨大
- 核销集中在当天（占核销的 68%），次日及以后衰减极快
- **策略**：领券当天推送提醒，过 24h 核销概率骤降

## 六、建议策略

### 6.1 优惠券设计
- **降低门槛**：≤20 元满减券核销率最高（14.42%），多推小额满减
- **提高折扣率**：折扣券比满减券核销率高，适当增加纯折扣券比例
- **当天有效**：当天核销占比 68%，可设置短有效期促进即时消费

### 6.2 推送策略
- **距离优先**：优先向距离 ≤3m 的用户推送（核销率 4%+）
- **工作日推送**：工作日核销量比周末高 21%
- **即时提醒**：领券后 2 小时内推送提醒，抓住当天核销窗口

### 6.3 用户运营
- **核心用户**（23.5%）：保持高频小额券，稳定粘性
- **流失用户**（23.8%）：推送大力度「回归券」，召回激励
- **新用户**（7.9%）：首单立减，降低首次核销门槛
